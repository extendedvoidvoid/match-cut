# SEE — video results (by lane)

Two products lived in one repo by accident of vibe coding. **Do not mix them.**

| Command | Opens | Product |
|---------|--------|---------|
| **`mc see`** | `SEE/` | **Essay / vertical** (Station F path, album-essay sibling) |
| **`mc see vj`** | `SEE/vj/` | **VJ** ~1 min montages / reels only |
| `mc see all` | `SEE/all/` | Both (debug only) |

```bash
mc see          # default — essay
mc see vj       # VJ experiments
mc see list     # essay tray, no Finder
```

Full map: `docs/history/LANES.md`

| Prefix | Lane |
|--------|------|
| `vertical__*` · `site__*` | Essay / Station F reframe |
| `reel__*` · `ref__*` | VJ only |

Real files stay under `exports/` and `public/`. `SEE/` is a tray of symlinks.
